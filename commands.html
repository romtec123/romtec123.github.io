2B2BOT Commands


/help

This command will send a link to this website.


/info

This command gives some info about the bot and our privacy policy.


/config

This command allows discord admins to setup 2b2bot chat, joins and famous joins channels.


/configs

This command shows all of the configured channels and settings.


/lookup - /profile - /msgs - /seen - /stats - /joindate - /kills - /deaths - /lastseen - /joins - /leaves

This command and it's aliases show data that was collected of a player.


/queue - /prio - /tps

This command and it's aliases show data about 2b2t including playercount, normal and prio queue lengths and tps.


/restarts

This command and its sub-commands allows you to set the restart notification role, channel and create an embed to give users the role.


/stalker

This command allows you to add, remove and list to your stalker list. When someone joins on your list, you will be DM'ed by the bot.
